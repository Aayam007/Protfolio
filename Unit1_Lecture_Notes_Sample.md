## 1.1 Introduction to .NET Framework (Syllabus Topic 1.1)

### What is .NET?

**.NET** is a free, open-source, cross-platform framework for building modern applications. It was developed by Microsoft and provides a comprehensive programming model for building various types of applications.

**Key Components of .NET:**

1. **Common Language Runtime (CLR)** - The execution engine
2. **Base Class Library (BCL)** - Collection of reusable classes
3. **Framework Class Library (FCL)** - Extensive library for application development

### .NET Ecosystem Evolution

| Framework | Description | Platform | Current Status |
|-----------|-------------|----------|----------------|
| **.NET Framework** | Original Windows-only framework | Windows | Maintenance mode |
| **.NET Core** | Cross-platform, open-source rewrite | Windows, Linux, macOS | Evolved to .NET 5+ |
| **.NET 5/6/7/8+** | Unified platform (successor to .NET Core) | Cross-platform | Active development |
| **Mono** | Open-source implementation | Multiple platforms | Used for Xamarin |

### Architecture of .NET

```
┌─────────────────────────────────────────────┐
│         Application Layer                    │
│  (Console, Web, Desktop, Mobile Apps)       │
└─────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────┐
│    Framework Class Library (FCL)            │
│  (System.*, ASP.NET Core, EF Core, etc.)   │
└─────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────┐
│    Common Language Runtime (CLR)            │
│  (Memory Management, Security, etc.)        │
└─────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────┐
│         Operating System                     │
│    (Windows, Linux, macOS)                  │
└─────────────────────────────────────────────┘
```

## 1.2 Compilation and Execution of .NET Applications (Syllabus Topic 1.2)

### The Compilation Process

**.NET uses a two-stage compilation process:**

#### Stage 1: Source Code → MSIL (Microsoft Intermediate Language)
```
C# Source Code (.cs) 
    ↓
C# Compiler (Roslyn)
    ↓
MSIL/CIL Code (.dll or .exe)
    +
Metadata (type information)
```

#### Stage 2: MSIL → Native Code (JIT Compilation)
```
MSIL Code
    ↓
Just-In-Time (JIT) Compiler
    ↓
Native Machine Code
    ↓
Execution by CPU
```

### Key Terms (Syllabus Requirement)

**CLI (Common Language Infrastructure):**
- Open specification developed by Microsoft
- Describes executable code and runtime environment
- Allows multiple programming languages to run on .NET

**MSIL (Microsoft Intermediate Language):**
- Also called CIL (Common Intermediate Language) or IL
- CPU-independent code generated by .NET compilers
- Similar to assembly language but platform-independent
- Contains metadata about types, members, and references

**CLR (Common Language Runtime):**
- Execution engine for .NET applications
- Provides services: memory management, type safety, exception handling, garbage collection

### Advantages of Two-Stage Compilation

1. **Platform Independence** - MSIL can run on any platform with CLR
2. **Language Interoperability** - Multiple languages compile to same MSIL
3. **Performance** - JIT optimization for specific hardware
4. **Security** - Code verification before execution

### Example: Viewing MSIL Code

```bash
# Compile C# code
dotnet build

# View MSIL using ildasm (Windows) or dotnet-ildasm
ildasm Program.dll
```

## 1.3 Basic Language Constructs in C# (Syllabus Topic 1.3)

### C# Program Structure

```csharp
using System;  // Namespace import

namespace MyApplication  // Namespace declaration
{
    // Class declaration
    class Program
    {
        // Main method - entry point
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, .NET Core!");
        }
    }
}
```

### Data Types in C#

**Value Types:**
```csharp
// Integral types
int age = 25;              // 32-bit integer
long population = 8000000000L;  // 64-bit integer
byte level = 255;          // 8-bit unsigned

// Floating-point types
float price = 99.99f;      // 32-bit floating point
double pi = 3.14159;       // 64-bit floating point
decimal salary = 50000.50m;  // 128-bit precise decimal

// Other value types
bool isStudent = true;     // Boolean
char grade = 'A';          // Character
```

**Reference Types:**
```csharp
string name = "John Doe";  // String (reference type)
object obj = new object(); // Base type for all types
int[] numbers = {1, 2, 3}; // Arrays (reference type)
```

### Variables and Constants

```csharp
// Variable declaration
int score;        // Declaration
score = 100;      // Assignment
int total = 50;   // Declaration + Initialization

// Constants (cannot be changed)
const double PI = 3.14159;
const int MAX_STUDENTS = 50;

// var keyword (implicit typing)
var message = "Hello";  // Compiler infers string type
var count = 10;         // Compiler infers int type
```

### Operators

```csharp
// Arithmetic operators
int sum = 10 + 5;      // Addition: 15
int diff = 10 - 5;     // Subtraction: 5
int product = 10 * 5;  // Multiplication: 50
int quotient = 10 / 5; // Division: 2
int remainder = 10 % 3; // Modulus: 1

// Comparison operators
bool isEqual = (10 == 10);     // Equal to: true
bool isNotEqual = (10 != 5);   // Not equal: true
bool isGreater = (10 > 5);     // Greater than: true
bool isLess = (5 < 10);        // Less than: true

// Logical operators
bool andResult = true && false;  // AND: false
bool orResult = true || false;   // OR: true
bool notResult = !true;          // NOT: false

// Assignment operators
int x = 10;
x += 5;   // x = x + 5;  (15)
x -= 3;   // x = x - 3;  (12)
x *= 2;   // x = x * 2;  (24)
```

### Control Structures

**Conditional Statements:**
```csharp
// if-else statement
int score = 85;
if (score >= 90)
{
    Console.WriteLine("Grade: A");
}
else if (score >= 80)
{
    Console.WriteLine("Grade: B");
}
else
{
    Console.WriteLine("Grade: C or below");
}

// switch statement
int day = 3;
switch (day)
{
    case 1:
        Console.WriteLine("Monday");
        break;
    case 2:
        Console.WriteLine("Tuesday");
        break;
    case 3:
        Console.WriteLine("Wednesday");
        break;
    default:
        Console.WriteLine("Other day");
        break;
}
```

**Loops:**
```csharp
// for loop
for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"Iteration: {i}");
}

// while loop
int count = 0;
while (count < 5)
{
    Console.WriteLine($"Count: {count}");
    count++;
}

// do-while loop
int num = 0;
do
{
    Console.WriteLine($"Number: {num}");
    num++;
} while (num < 5);

// foreach loop (for collections)
int[] numbers = {1, 2, 3, 4, 5};
foreach (int n in numbers)
{
    Console.WriteLine(n);
}
```

## 1.4 Object-Oriented Programming in C# (Syllabus Topics 1.4-1.6)

### Classes and Objects

**Class Definition:**
```csharp
// Class - Blueprint for objects
public class Student
{
    // Fields (data members)
    private string name;
    private int age;
    private string rollNumber;
    
    // Constructor - initializes object
    public Student(string name, int age, string rollNumber)
    {
        this.name = name;
        this.age = age;
        this.rollNumber = rollNumber;
    }
    
    // Methods (behavior)
    public void DisplayInfo()
    {
        Console.WriteLine($"Name: {name}, Age: {age}, Roll: {rollNumber}");
    }
}

// Creating objects
Student student1 = new Student("John", 20, "S001");
Student student2 = new Student("Jane", 21, "S002");

student1.DisplayInfo();  // Output: Name: John, Age: 20, Roll: S001
```

### Constructors (Syllabus Topic)

**Types of Constructors:**

```csharp
public class Book
{
    public string Title { get; set; }
    public string Author { get; set; }
    public decimal Price { get; set; }
    
    // Default constructor
    public Book()
    {
        Title = "Unknown";
        Author = "Unknown";
        Price = 0.0m;
    }
    
    // Parameterized constructor
    public Book(string title, string author, decimal price)
    {
        Title = title;
        Author = author;
        Price = price;
    }
    
    // Constructor overloading
    public Book(string title, string author)
    {
        Title = title;
        Author = author;
        Price = 0.0m;  // Default price
    }
    
    // Copy constructor
    public Book(Book otherBook)
    {
        Title = otherBook.Title;
        Author = otherBook.Author;
        Price = otherBook.Price;
    }
}

// Usage
Book book1 = new Book();                          // Default constructor
Book book2 = new Book("C# Guide", "Mark Price", 49.99m);  // Parameterized
Book book3 = new Book("ASP.NET", "Andrew Lock");  // Overloaded
Book book4 = new Book(book2);                     // Copy constructor
```

### Properties (Syllabus Topic)

**Property Syntax:**

```csharp
public class Person
{
    // Full property with backing field
    private string _name;
    public string Name
    {
        get { return _name; }
        set 
        { 
            if (string.IsNullOrWhiteSpace(value))
                throw new ArgumentException("Name cannot be empty");
            _name = value; 
        }
    }
    
    // Auto-implemented property (simplified)
    public int Age { get; set; }
    
    // Read-only property
    public string FullName { get; }
    
    // Property with private setter
    public DateTime CreatedDate { get; private set; }
    
    // Calculated property (no backing field)
    public bool IsAdult
    {
        get { return Age >= 18; }
    }
    
    public Person(string firstName, string lastName)
    {
        FullName = $"{firstName} {lastName}";
        CreatedDate = DateTime.Now;
    }
}

// Usage
Person person = new Person("John", "Doe");
person.Name = "John Doe";
person.Age = 25;
Console.WriteLine(person.IsAdult);  // True
```

## 1.5 Arrays and Strings (Syllabus Topic)

### Arrays

**Single-Dimensional Arrays:**
```csharp
// Declaration and initialization
int[] numbers = new int[5];  // Array of 5 integers
numbers[0] = 10;
numbers[1] = 20;

// Inline initialization
int[] scores = {90, 85, 92, 88, 95};

// Using array elements
for (int i = 0; i < scores.Length; i++)
{
    Console.WriteLine($"Score {i + 1}: {scores[i]}");
}

// foreach loop
foreach (int score in scores)
{
    Console.WriteLine(score);
}
```

**Multi-Dimensional Arrays:**
```csharp
// 2D Array (matrix)
int[,] matrix = new int[3, 3]
{
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Accessing elements
Console.WriteLine(matrix[0, 0]);  // Output: 1
Console.WriteLine(matrix[1, 2]);  // Output: 6

// Iterating through 2D array
for (int i = 0; i < matrix.GetLength(0); i++)
{
    for (int j = 0; j < matrix.GetLength(1); j++)
    {
        Console.Write($"{matrix[i, j]} ");
    }
    Console.WriteLine();
}
```

**Jagged Arrays:**
```csharp
// Array of arrays
int[][] jaggedArray = new int[3][];
jaggedArray[0] = new int[] {1, 2};
jaggedArray[1] = new int[] {3, 4, 5};
jaggedArray[2] = new int[] {6, 7, 8, 9};

// Accessing jagged array
Console.WriteLine(jaggedArray[1][2]);  // Output: 5
```

### Strings

**String Operations:**
```csharp
// String creation
string str1 = "Hello";
string str2 = "World";

// Concatenation
string greeting = str1 + " " + str2;  // "Hello World"
string message = string.Concat(str1, " ", str2);

// String interpolation (C# 6+)
string name = "John";
int age = 25;
string info = $"Name: {name}, Age: {age}";

// String methods
string text = "  C# Programming  ";
Console.WriteLine(text.Length);         // 19
Console.WriteLine(text.Trim());         // "C# Programming"
Console.WriteLine(text.ToUpper());      // "  C# PROGRAMMING  "
Console.WriteLine(text.Substring(5, 12)); // "Programming"
Console.WriteLine(text.Contains("C#"));  // True
Console.WriteLine(text.Replace("C#", "Java")); // "  Java Programming  "

// String splitting
string csv = "John,25,Engineer";
string[] parts = csv.Split(',');
// parts[0] = "John", parts[1] = "25", parts[2] = "Engineer"

// String comparison
string a = "hello";
string b = "HELLO";
Console.WriteLine(a == b);                          // False (case-sensitive)
Console.WriteLine(a.Equals(b, StringComparison.OrdinalIgnoreCase));  // True

// StringBuilder for efficient string manipulation
using System.Text;
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 1000; i++)
{
    sb.Append(i);  // More efficient than string concatenation
}
string result = sb.ToString();
```

## Summary

**Key Takeaways:**
- .NET provides a unified framework for building cross-platform applications
- C# code compiles to MSIL, which is then JIT-compiled to native code
- C# supports both value types (stored on stack) and reference types (stored on heap)
- Properties provide controlled access to class fields with validation
- Constructors initialize objects; can be overloaded for flexibility
- Arrays store multiple values of the same type; support multi-dimensional structures
- Strings are immutable; use StringBuilder for efficient manipulation
